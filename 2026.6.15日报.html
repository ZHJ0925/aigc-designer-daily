01 观点 优设网 今天

用 GPT Image 2 做完一个真实 UI 项目后，我发现 AI 最难的不是出图

GPT Image 2 能快速生成惊艳概念图，但信息层级不清、多轮修改易失控、无法直接用于前端还原——AI 出图 ≠ 会做 UI，设计师需从执行转向架构与判断。

核心工作流启示：先分析需求 → 再找参考 → 让 AI 发散多方案 → 最终由设计师收敛判断。AI 加速的是方案探索，不是替代决策。

02 观点 优设网 今天

做了多个爆款 Skills 以后，我发现它可能是 Agent 时代最大的能力商品

Skill 是将专家经验、工作流和品味封装成可分发的 Agent 能力单元，不仅是提示词，更包含规则、流程和失败经验。设计师可从执行者进化为「能力架构师」。

将十年设计经验编码为 Skill 是 Agent 时代建立个人品牌的关键路径——记录失败案例（Gotchas）比成功流程更有价值，因为负面约束才真正提升输出稳定性。

03 观点 优设网 昨天

WWDC26 后，Apple 系统图标发生了哪些变化？

iOS 27 和 macOS Golden Gate 的 36 款原生图标迎来 Liquid Glass 进化：颜色更大胆、玻璃质感变轻变平、细节更锐利干净，整体从「厚重折射」转向「轻盈清晰」。

Apple 的「视觉减法」趋势值得关注：简化符号、强化对比、色彩分区代替模糊透明。这套跨平台统一语言将影响下一代 UI 设计范式。
